# Chapter1
1. 现代气候概念的要点是什么
	- 现代气候概念：<span class="cloze-span">气候系统在一个时间段内的平均状态及其变化或变率</span>。一般可用气候系统的平均值和高阶矩统计量（例如：方差，协方差等）来表示
	- 气候状态：气候系统的全部成分在任一特定时段内的平均统计特征
	- 气候变化：指气候平均状态（平均值）和离差（方差）两者中的一个或两个一起出现了统计意义上显著的变化
2. 说明气候系统的组成及其主要性质
	- 定义：<span class="cloze-span">气候系统由大气圈、水圈、冰雪圈、岩石圈和生物圈及其相互作用组成的高度复杂系统</span>
	- 大气圈：指均匀分布在地球表面上薄薄的一层（地球半径的0.5%）气体混合物。多与**动力学**有关，而大气以外的其他成员， 如海洋、冰雪、陆面等对大气的影响则主要是热力作用
	- 水圈：由地球上的所有液态水构成，包括海洋、内海、湖泊、河流及地下水。对于气候研究，海洋是水圈中最重要的，它们覆盖了地球表面约三分之二的区域。主要有三个作用：<span class="cloze-span">大气运动重要的热源和水汽源地；极大的热容量存储大气运动信号；二氧化碳最大的汇</span>
	- 冰雪圈：由地球表面积存的大量的雪和冰构成，包括格陵兰和南极的大陆冰原，以及其他的大陆冰川、雪原、海冰等，是地球上最大的淡水储存库。主要特点是<span class="cloze-span">对太阳辐射的高反射率；很低的热传导性；明显的季节性和年际变化</span>
	- 岩石圈：地球表层的固体表面，由岩石、土壤和沉积物组成，包括高原、平原、山地、丘陵、盆地等。主要特点是<span class="cloze-span">热容量小热惯性小；动力作用（爬越作用；绕流作用）；热力作用（夏季热源，冬季冷源）</span>
	- 生物圈：由地面植被、大陆动物群、海洋的植物及动物群构成（包括人类自身）。主要作用是<span class="cloze-span">植被改变了地表粗糙度、地表反射率、蒸发、径流及土壤比热容；生物可以影响大气和海洋的CO2平衡以及气溶胶的产生；工农业、城市化等人类活动会与气候系统发生相互作用</span>
	- 气候系统的性质：<span class="cloze-span">气候系统是一个复杂的、高度非线性的、开放的巨系统</span>。全球性的非孤立系统；响应时间差异很大，可分为内部系统和外部系统；不稳定的高度耗散系统
	- 气候系统的反馈过程
		- 反馈：气候系统不同属性（变量）之间的相互作用，引起气候属性的变化，称为反馈，包括正反馈过程和负反馈过程
		- 正反馈：反馈过程造成的气候变化与原变化同号，使气候变化加剧，产生气候不稳定，称为正反馈。如<span class="cloze-span">冰雪量-辐射反照率-温度</span>：升温→冰雪融化→太阳辐射地表反照率降低，地表吸收更多热量→温度持续升高；<span class="cloze-span">水汽-红外逸出辐射-温度</span>：水汽增加→温室效应加强→陆地和海洋温度上升→地表蒸发加剧，产生更多水汽
		- 负反馈：反馈过程造成的气候变化与原变化反号，抑制气候的变化和异常，使气候趋于稳定，称为负反馈。如<span class="cloze-span">赤道极地温差-热量输送</span>：赤道与两极之间温差大→赤道向两极的热量输送量增加→赤道与两极间温差减小；<span class="cloze-span">水汽蒸发量-水面温度</span>：水汽蒸发量大→水面温度降低→水汽蒸发量减小
3. 什么是气候变化？气候变化的原因有哪些？气候变化的特征有哪些
	- 由各种要素（气温、降水、气压等）所表征的气候状态相对于某一气候标准态的偏差或同类气候状态间的变化，称为气候变化；而当这种偏差（变化）超过一定程度，则称为气候异常
	- 气候变化：<span class="cloze-span">指气候平均状态（平均值）和离差（方差）两者中的一个或两个一起出现了统计意义上显著的变化</span>
	- 原因
		- 外部因子：外部因子不受或者基本不受气候系统状况的影响，气候系统对这些没有反馈作用
			- 一般性的因子（如太阳辐射、地球轨道要素等）
			- 区域性和局地性的因子（如海洋、地形、植被等）
		- 内部因子：内部因子与系统内部不同物理过程中的相互作用有关，包括正反馈负反馈
	- 特征：<span class="cloze-span">气候变化的多时间尺度性</span>（就比如短期气候变化月季年、中期年际、长期年代际……）；<span class="cloze-span">阶段性</span>（冰期间冰期、亚冰期亚间冰期云云）；<span class="cloze-span">突变性</span>（包括均值突变、趋势突变、变率突变，其中变率突变包括振幅突变和频率突变）
4. 气候可预报性怎样分类
	- 第一类可预报性：<span class="cloze-span">初始误差（扰动）随时间增长问题（确定性预报的时效问题）</span>
	- 第二类可预报性：<span class="cloze-span">外强迫变化引起气候变化的模拟和预报能力（对外强迫的响应及敏感性）</span>
5. 什么是短期气候预测？短期气候预测的方法有几类
	- 短期气候预测：一般指月到季尺度的预测
	- 方法
		- 物理统计方法（也称为经验方法）
		- 动力数值方法
		- 动力-统计相结合的方法


# Chapter2 大气环流的基本状况
1. **控制大气环流的主要因子**
	- 大气环流的定义：大气环流是指大范围(水平尺度几千公里以上)较长时间尺度的(几天以上)大气运动的基本状况。
	- <span class="cloze-span">大气本身的特殊尺度</span>：薄层近似；垂直速度量级小；准水平
	- <span class="cloze-span">太阳辐射随纬度分布的不均匀性</span>：辐射差额的梯度在两个半球的中高纬度附近达到最大，因此温度水平梯度、急流在中纬度最大。
	- <span class="cloze-span">地球自转</span>：三风四带和三圈环流的形成；减小大气不稳定性；减小大气位能的释放
	- <span class="cloze-span">地球表面的不均匀性</span>：海陆分布产生的不均匀加热；中高纬温度梯度；大地形作用（动力、热力）
	- <span class="cloze-span">地面摩擦</span>：高空槽脊向西倾斜；大气能量平衡，对大气角动量的制造和平衡具有重要作用。
	- <span class="cloze-span">大气内部动力学过程和非线性相互作用</span>
	- 前三者：大气环流的准水平性、准地转性、经向三圈环流、东西风带的维持；地球表面的不均匀性有利于经向环流分量；地面摩擦的作用有利于东西风带的维持。
2. **说明冬夏季海平面气压场、对流层中部位势高度场和平流层底部位势高度场的主要差异有哪些。形成原因是什么？**
	- 海平面气压场
		- <span class="cloze-span">北半球中高纬：冬季大气活动中心有阿留申低压、冰岛低压、蒙古高压和加拿大高压。夏季为太平洋副高、大西洋副高、南亚热低压、北美热低压和冰岛低压</span>
		- <span class="cloze-span">低纬度：赤道辐合带</span>
		- <span class="cloze-span">南半球中高纬：南太平洋副高、南大西洋副高和印度洋高压</span>
	- 对流层中部
		- <span class="cloze-span">槽脊：冬季呈三槽（亚洲大陆东岸、北美大陆东岸和乌拉尔山以西）三脊（阿拉斯加、西欧沿岸和贝加尔湖地区）型；夏季呈四槽（东亚大槽东移至勘察加半岛，北美大槽，西欧沿岸，乌拉尔山以东）四脊型</span>
		- <span class="cloze-span">极涡：极涡的中心都不在南北极。 1月北半球的极涡有两个中心，7月只有一个中心。南半球极涡无论冬夏都只有一个中心</span>
		- <span class="cloze-span">急流：1月平均最大地转西风轴线比7月偏南偏强</span>
		- <span class="cloze-span">副高：夏季比冬季显著偏强偏北</span>
	- 平流层底部
		- <span class="cloze-span">大气长波：冬季100hPa面上的西风带准静止波波长增长，波数减少，一般呈2波型。夏季西风环流显著减弱，准静止波的振幅显著变小</span>
		- <span class="cloze-span">南亚高压：夏季南亚上空的反气旋显著增强。其中心位置冬季位于我国南海上空；夏季青藏高原上空，西起大西洋，横跨亚非大陆，东至西南太平洋</span>
3. **平流层大气环流的主要特点有哪些？**
	- <span class="cloze-span">平流层中部大气环流由冬到夏从极涡转变为绕极反气旋，偏西风到偏东风</span>
	- <span class="cloze-span">整个北半球的水平温度梯度由冬到夏发生了方向相反的转变</span>
4. **亚洲季风的主要特征是什么？**
	- 季风：一般将季风定义为：一个地区冬、夏的盛行风向有明显的季节变化，盛行风向的夹角达120º-180º
	- 因素：<span class="cloze-span">海陆热力差异；大尺度行星环流的南北位移；高原大地形的影响</span>
	- <span class="cloze-span">南亚和西太平洋季风区是热带季风区，而东亚季风区是一种由热带与副热带和中高纬度系统相互作用构成的季风区</span>
	- 南亚季风：<span class="cloze-span">夏季盛行来自印度洋和阿拉伯海的西南季风；冬季盛行来自亚洲大陆的东北季风</span>
	- 东亚夏季风：<span class="cloze-span">北进过程非连续（三次北跳和四次停滞）；南撤迅速</span>
	- 东亚冬季风：<span class="cloze-span">其盛行风向在我国华北及日本北部和中部为西北风，我国黄河以南和日本南部为东北风</span>
5. **大气环流的表征方法有哪些？**
	- 定性特征表述
		- 分型法：对大气环流或气象要素的演变特征进行分析，把特征相似的个例归并为一类
		- 相似法：当前及其此前的大气环流特征与历史同期状况相似的话，其未来演变也应该与历史相似，从而按照历史状况做出预测
		- 模态法：利用如EOF分解计算得到若干气象场的正交函数
	- 大气环流指数
	- 平均图（滤除移动性短波系统，突出长波和超长波的特点）、距平图（给定时段的平均图减去同时段的多年平均图所得的差值图）、纬偏图（给定时段的平均图减去同时段的纬圈平均值所得的差值图）
	- 典型场分析，如EOF/SVD


# Chapter3 大气低频变化及其遥相关
1. **大气运动按照时间尺度划分，可以分为几类？**
	- 大气运动可以分为：年代际、年际、季节内、天气、短时间
	- 大气变化
		- <span class="cloze-span">大气低频变化</span>：时间尺度为10d以上的变化
		- <span class="cloze-span">天气尺度变化</span>：时间尺度为2.5～6d的变化
		- <span class="cloze-span">高频变化</span>：时间尺度小于2d的变化
2. **如何利用大气原始资料提取资料中低频变化的部分**：<span class="cloze-span">使用数字滤波器</span>：Blackmon滤波器；逐日资料31点低通（低频通过）滤波器；Butterworth带通滤波器
3. **大气低频变化的地理分布具有哪些特点？**（具体情况具体分析吧）
	- 以**冬季500hPa位势高度均方差**分布为例，通过比较原始资料、高通（10d以下）滤波资料、低通（10d以上）滤波资料、30d平均（带通滤波）资料
		- 高通资料在45N附近的大西洋和太平洋各有一个拉长了的极大值，它们与风暴轴有关，该处斜压波活动最强
		- 低通资料高值中心分别位于北大西洋、北太平洋和西伯利亚北极区。这三个方差高值区与阻塞形势频繁活动区相连接
		- **大气低频变化对大气变化方差贡献最大**
	- 40～50天振荡统计显著区域的地理分布主要位于欧亚大陆和北美大陆的东部并延伸到邻近大陆的海洋上；这两个区域分别与北半球的两个大槽有关，位于风暴轴的西南方向，说明40-60天振荡与大气角动量从较低纬度向较高纬度输送有关。
4. **什么是大气遥相关？可以利用哪些方法研究大气遥相关？**
	- <span class="cloze-span">大气遥相关是指相隔一定时间和空间的气象变量或者天气气候过程之间稳定相关的地理分布型</span>。包括同一变量空间同一时间的、不同变量空间同一时间的、同一变量空间不同时间的、不同变量空间不同时间的
	- 方法：<span class="cloze-span">单点相关系数</span>，即计算某一基本点的要素资料序列与其它点的资料序列之间的同时相关系数，相关系数的地理分布就给出一种不同区域的相关特点，然后再对相关系数做比如显著性检验，得到与该点显著相关的区域。定义为遥相关型。遥相关型表征了大气低频变化的水平结构。
5. **冬季北半球海平面气压有哪些遥相关型？**
	- <span class="cloze-span">NAO（北大西洋涛动North Atlantic Oscillation）：冰岛低压和亚速尔高压之间</span>；冰岛低压加深，则亚速尔高压增强。
	- <span class="cloze-span">NPO（北太平洋振荡North Pathetic Oscillation）：阿留申低压和太平洋高压</span>
	- <span class="cloze-span">SO（南方涛动Southern Oscillation）</span>：东南太平洋与印度洋及印尼地区之间的反相气压振动
	- <span class="cloze-span">AO（北极涛动Arctic Oscillation；NAM，北半球环状模）：北半球极地地区和北半球中高纬度地区的海平面气压之间存在南北方向的反相关</span>
	- <span class="cloze-span">AAO（南极涛动）</span>：南半球极地地区和南半球中高纬度地区的海平面气压之间反相关
6. **500hPa高度场有哪些遥相关？**
	- 北半球冬季
		- <span class="cloze-span">WA（西大西洋型）：大西洋西部副热带高压和冰岛低压之间的反相关。</span>
		- <span class="cloze-span">EA（大西洋东部型）：在非洲以西的大西洋东部与英国以西的大西洋东部之间反相关，而与欧洲上空的位势高度正相关。</span>
		- <span class="cloze-span">EU（欧亚型）：西欧上空位势高度与西伯利亚之间反相关，而与中国东北和日本一带则为正相关。</span>
		- <span class="cloze-span">WP（太平洋西部型）：当西太平洋副高加强时，阿留申低压深，表现出反相关的特点。</span>
		- <span class="cloze-span">PNA（太平洋北美型）：表现为热带和副热带太平洋高压加强，位于阿留申地区的低压加深，位于北美西北部的脊加强，美国东部的槽加深，造成美国冬季严寒天气。</span>
	- 北半球夏季
		- <span class="cloze-span">EAP（==东亚-太平洋==遥相关关系）：它表示了从菲律宾沿海-我国东部和日本沿海-白令海-阿拉斯加-太平洋北美沿海一系列的遥相关型波列。</span>
		- SRP（丝绸之路遥相关）：对流层高层沿亚洲西风急流传播的波列，表现为沿亚洲急流交错分布的南风和北风异常（或气旋反气旋式环流异常）
		- BBC（英国-贝加尔湖遥相关，代表==极锋急流==）：气旋，冷湿；反气旋，暖干。
7. 其他时间尺度的低频变化
	- 大气季节内振荡MJO
	- 准两年振荡TBO，如季风环流、降水、 海温等具有2～3 年周期的年际变化称之为对流层准两年振荡（ TBO）
	- 大气中的韵律：指两种天气现象或天气过程相距一定的时间间隔发生重演的规律性
		- 两种天气现象或者天气过程之间的时间间隔具有准稳定性
		- 两次天气现象或者天气过程之间的中间过程并不清楚，不可能从开始的现象或过程追踪到对应的现象或者过程
		- 韵律一般只出现在固定季节
8. 大气低频变化的可能原因
	- 外因：大气对外源强迫的响应。如海表温度异常、土壤湿度异常、海冰异常……
	- 内因：大气内部动力学过程的作用。如振荡线性、波动相互作用、反馈过程……


# Chapter4 海气相互作用
1. 海气相互作用的基本物理过程是什么?
	- 海洋对大气主要是热力的，主要通过海洋向大气的感热和潜热输送
	- 大气对海洋主要是动力的，主要通过风场的牵引、洋流的上翻以及对辐射过程的影响
2. 低纬度地区海气相互作用与中高纬度地区的海气相互作用有何差异。
	- 热带大气的大尺度运动基本上是对海洋加热的响应，而次表层以上的海洋的运动则是对大气风应力的响应
	- 中高纬度的海气相互作用主要表现为大气对海洋的强迫
3. 什么是厄尔尼诺和拉尼娜事件?厄尔尼诺事件的形成机制是什么?
	- 厄尔尼诺：赤道中东太平洋海面温度大范围持续异常升高
	- 拉尼娜：赤道中东太平洋海表温度在冬季出现较强的负距平现象
	- 厄尔尼诺的形成机制暂无定论，目前的理论有充放电理论、延迟振子理论。
4. 简述厄尔尼诺、南方涛动与Walker环流的关系
	- 沃克环流认为，赤道信风吹到西太平洋暖海水区域辐合上升，上升气流在高层辐散，产生一支向东的回流，这支回流在东太平洋下沉，在热带太平洋形成纬向的垂直环流。
	- 一般情况下盛行的东信风使太平洋海温东高西低，同时温跃层是西高东低。一旦东信风减弱，则西太暖池盈余的暖海水便会西传，使东太平洋海表温度增加，可能导致厄尔尼诺的出现。
	- 而厄尔尼诺的出现又往往与南方涛动的负位相相关，即印度洋印尼澳大利亚海平面气压上升，中太平洋大溪地海平面气压下降。
5. 何谓ENSO?它对全球大气环流和天气气候异常有何影响?
	- 厄尔尼诺对应的暖水位向与拉尼娜对应的冷水位相，以及太平洋与印度洋海平面气压场的反位相变化SO重复非周期性的出现构成了ENSO。
	- 对西太暖池的影响，进而影响东亚夏季风、西太台风生成等
	- 对印度洋海温的影响，热带印度洋全区一致海温模态是热带印度洋海温变化的最主要模态，一般认为热带印度洋全区一致增暖模态是对厄尔尼诺的滞后响应；以及对印度洋偶极子模态的影响
	- 西北太平洋海温的影响，对黑潮暖流的影响，进而对我国夏季降水的影响
